# Robot Framework Integration

## Overview

RPAForge is built on Robot Framework and generates standard, readable Robot Framework syntax. This document covers the integration details and best practices.

## Robot Framework Version

**Minimum Required**: Robot Framework 5.0+

Features used:
- `TRY/EXCEPT/FINALLY` (RF 5.0+)
- `WHILE` loops (RF 5.0+)
- Inline `IF` (RF 5.0+)
- `RETURN` statement (RF 5.0+)

## Code Generation

### File Structure

```robotframework
*** Settings ***
Documentation     Generated by RPAForge
Library           SeleniumLibrary
Library           RPA.Desktop
Resource          ../variables/default.py

*** Variables ***
${URL}            https://example.com
${BROWSER}        chrome

*** Keywords ***
Login Flow
    [Arguments]    ${username}    ${password}
    Open Browser    ${URL}    ${BROWSER}
    Input Text    id=username    ${username}
    Click Button    id=login
    [Return]    ${True}

*** Tasks ***
Main Process
    Login Flow    ${USERNAME}    ${PASSWORD}
    Click Element    id=dashboard
```

### Block to Keyword Mapping

| Block | Robot Framework |
|-------|-----------------|
| Start | `*** Tasks ***` section start |
| End | End of task |
| If | `IF ... ELSE ... END` |
| While | `WHILE ... END` |
| For Each | `FOR ... IN ... END` |
| Try Catch | `TRY ... EXCEPT ... END` |
| Parallel | `Run Parallel` |
| Sub-Diagram Call | User keyword call |

## Control Structures

### IF Statement

**Visual:**
```
┌─────────────────────────────────────┐
│ IF                                  │
│ Condition: ${status} == "success"   │
├─────────────────┬───────────────────┤
│ TRUE            │ FALSE             │
│ [activities]    │ [activities]      │
└─────────────────┴───────────────────┘
```

**Generated Code:**
```robotframework
IF    ${status} == "success"
    Log    Operation succeeded
    Send Notification    success
ELSE
    Log    Operation failed
    Send Notification    failure
END
```

**ELSE IF Chain:**
```robotframework
IF    ${status} == "success"
    Log    Success
ELSE IF    ${status} == "warning"
    Log    Warning
ELSE IF    ${status} == "error"
    Log    Error
ELSE
    Log    Unknown status
END
```

### WHILE Loop

**Visual:**
```
┌─────────────────────────────────────┐
│ WHILE                               │
│ Condition: ${counter} < 10          │
│ Limit: 100                          │
├─────────────────────────────────────┤
│ [Loop Body]                         │
└─────────────────────────────────────┘
```

**Generated Code:**
```robotframework
WHILE    ${counter} < 10    limit=100
    Log    Counter: ${counter}
    ${counter}=    Evaluate    ${counter} + 1
END
```

### FOR Loop

**Visual:**
```
┌─────────────────────────────────────┐
│ FOR EACH                            │
│ Item: ${item}                       │
│ In: @{items}                        │
├─────────────────────────────────────┤
│ [Loop Body with ${item}]            │
└─────────────────────────────────────┘
```

**Generated Code:**
```robotframework
FOR    ${item}    IN    @{items}
    Log    Processing: ${item}
    Process Item    ${item}
END
```

**FOR IN RANGE:**
```robotframework
FOR    ${i}    IN RANGE    0    10
    Log    Index: ${i}
END
```

**FOR IN ENUMERATE:**
```robotframework
FOR    ${index}    ${item}    IN ENUMERATE    @{items}
    Log    ${index}: ${item}
END
```

### TRY/EXCEPT

**Visual:**
```
┌─────────────────────────────────────┐
│ TRY                                 │
│ [Try activities]                    │
├─────────────────────────────────────┤
│ EXCEPT ConnectionError AS ${err}    │
│ [Handler activities]                │
├─────────────────────────────────────┤
│ FINALLY                             │
│ [Cleanup activities]                │
└─────────────────────────────────────┘
```

**Generated Code:**
```robotframework
TRY
    Open Connection    ${host}
    Login    ${user}    ${pass}
EXCEPT    ConnectionError    AS    ${err}
    Log    Connection failed: ${err}
    Send Alert    Connection error
EXCEPT    AS    ${err}
    Log    Unexpected error: ${err}
    Send Alert    Unknown error
FINALLY
    Close Connection
END
```

## Variables

### Variable Types

```robotframework
*** Variables ***
# Scalar (string, number, boolean)
${STRING}         Hello, World!
${NUMBER}         ${42}
${BOOLEAN}        ${True}

# List
@{LIST}           item1    item2    item3

# Dictionary
&{DICT}           key1=value1    key2=value2

# Environment
${HOME}           %{HOME}
```

### Variable Scopes

```robotframework
*** Keywords ***
Example Keyword
    # Local variable (default)
    ${local}=    Set Variable    local value
    
    # Suite variable (global)
    Set Suite Variable    ${GLOBAL}    global value
    
    # Test variable (test scope)
    Set Test Variable    ${TEST_VAR}    test value
    
    # Global variable (available everywhere)
    Set Global Variable    ${ALL}    everywhere
```

### Custom Libraries

```python
from robot.api.deco import keyword, library

@library(scope="GLOBAL", auto_keywords=True)
class MyLibrary:
    """Custom RPAForge library."""
    
    @keyword(tags=["custom"])
    def do_something(self, arg: str) -> str:
        """Do something with the argument.
        
        :param arg: Input argument
        :returns: Processed result
        """
        return f"Processed: {arg}"
```

## Best Practices

### Naming Conventions

```robotframework
# Keywords - Title Case
Login To Application
Get User Data
Process Input File

# Variables - snake_case
${user_name}
${file_path}
${max_retry_count}

# Constants - UPPER_CASE
${URL}
${BROWSER}
${TIMEOUT}

# Lists - singular with @ prefix
@{user_list}
@{file_names}

# Dictionaries - descriptive with & prefix
&{config_data}
&{user_info}
```

### Documentation

```robotframework
*** Keywords ***
Process User Data
    [Documentation]    Processes user data from input file.
    ...    
    ...    Arguments:
    ...        - file_path: Path to input file
    ...        - output_dir: Output directory
    ...    
    ...    Returns:
    ...        - processed_count: Number of processed records
    ...    
    ...    Example:
    ...        ${count}=    Process User Data    input.xlsx    output/
    [Arguments]    ${file_path}    ${output_dir}
    # Implementation
    [Return]    ${processed_count}
```

### Error Handling

```robotframework
*** Keywords ***
Safe Operation
    [Documentation]    Performs operation with error handling.
    TRY
        ${result}=    Risky Operation
        Log    Success: ${result}
    EXCEPT    ValueError    AS    ${err}
        Log    Value error: ${err}
        ${result}=    Set Variable    ${None}
    EXCEPT    AS    ${err}
        Log    Unexpected error: ${err}
        Fail    Operation failed: ${err}
    FINALLY
        Cleanup Resources
    END
    [Return]    ${result}
```

## Performance

### Optimization Tips

1. **Use `Run Keyword If` for conditions**
```robotframework
# ✓ Good
Run Keyword If    ${condition}    Do Something

# ✗ Slower (creates branch)
IF    ${condition}
    Do Something
END
```

2. **Avoid deep nesting**
```robotframework
# ✓ Good - Use sub-diagrams
Validate Input
Process Data
Save Results

# ✗ Bad - Deep nesting
IF    ${valid}
    FOR    ${item}    IN    @{items}
        IF    ${item.active}
            # Deep nesting...
```

3. **Use built-in keywords**
```robotframework
# ✓ Good
${count}=    Get Length    ${list}

# ✗ Bad - Custom implementation
${count}=    Evaluate    len(${list})
```

4. **Batch operations**
```robotframework
# ✓ Good - Process all at once
${results}=    Process All Items    @{items}

# ✗ Bad - Process one by one
FOR    ${item}    IN    @{items}
    Process Item    ${item}
END
```

### Step-Through Debug

RPAForge debugger integrates with Robot Framework:

```python
from rpaforge.debugger import Debugger

# Set breakpoints
debugger.set_breakpoint("test.robot", line=15)

# Run with debug
for event in debugger.run_suite(suite):
    if event.type == "breakpoint_hit":
        # Show variables
        print(event.variables)
        # Wait for continue
        debugger.wait_for_continue()
```

## Continuous Integration

### Command Line Execution

```bash
# Run all tests
robot tests/

# Run specific test
robot tests/login.robot

# Run with tags
robot --include smoke tests/

# Run with variables
robot --variable URL:https://staging.example.com tests/

# Run with output
robot --output results.xml --log log.html --report report.html tests/
```

### Pabot (Parallel Execution)

```bash
# Install pabot
pip install robotframework-pabot

# Run tests in parallel
pabot tests/
```

### CI Configuration

```yaml
# .github/workflows/test.yml
name: Tests

on: [push, pull_request]

jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      
      - name: Set up Python
        uses: actions/setup-python@v4
        with:
          python-version: '3.11'
      
      - name: Install dependencies
        run: |
          pip install robotframework
          pip install robotframework-seleniumlibrary
          pip install rpaframework
      
      - name: Run tests
        run: robot --output results.xml tests/
      
      - name: Publish results
        uses: actions/upload-artifact@v3
        with:
          name: robot-results
          path: results.xml
```
